Política de Privacidade

Aplicativo Remédios — Última atualização: 15 de abril de 2026

1. Resumo

O Remédios é um aplicativo 100% offline para controle de estoque de medicamentos. Nenhum dado inserido no app é enviado a servidores nossos ou de terceiros. Toda informação fica armazenada exclusivamente no seu aparelho.

2. Dados que o app armazena

O app guarda localmente, apenas no seu aparelho, as informações que você mesmo insere:

  • Nomes de perfis
  • Nomes de medicamentos, dosagens, quantidades em estoque
  • Horários de lembrete e configurações de dose
  • Histórico de entradas e saídas de estoque

Esses dados ficam salvos apenas no seu aparelho, em uma área privada do aplicativo. Não temos acesso a eles. Não há conta de usuário, login, nem sincronização com a nuvem.

Exclusão dos dados: como todos os dados ficam apenas no aparelho, você pode removê-los a qualquer momento desinstalando o aplicativo ou usando "Limpar dados" nas configurações do Android.

3. Dados que o app NÃO coleta

  • Não coletamos dados pessoais identificáveis
  • Não usamos analytics, rastreamento ou publicidade
  • Não compartilhamos dados com terceiros
  • Não acessamos contatos, localização, câmera ou microfone
  • Não usamos conexão de rede para enviar dados

4. Backup e compartilhamento

Você pode exportar um arquivo .json com seus dados a qualquer momento pela tela Backup e Restauração. Esse arquivo é gerado localmente e só sai do seu aparelho se você escolher compartilhá-lo (por WhatsApp, e-mail, Drive, etc). O app não envia esse arquivo para nenhum servidor automaticamente.

5. Permissões do aparelho

  • Notificações — para lembrar de tomar medicamentos e alertar sobre estoque baixo
  • Alarmes exatos — para que os lembretes toquem exatamente no horário configurado
  • Iniciar com o sistema — para que os lembretes voltem a funcionar depois que o aparelho for reiniciado
  • Vibrar — para vibrar quando uma notificação toca
  • Manter o aparelho acordado (wake lock) — para garantir que a notificação toque mesmo com a tela desligada
  • Internet — embora essa permissão esteja declarada por uma exigência técnica do sistema usado para criar o app, o Remédios não acessa a internet, não envia nem recebe nada, e funciona totalmente offline

Nenhuma dessas permissões é usada para coleta ou transmissão de dados.

6. Crianças

O app não é direcionado nem coleta dados de menores de 13 anos (em conformidade com COPPA e LGPD). Como não coletamos dados, não há risco específico para crianças que usam o app sob supervisão adulta.

7. Alterações desta política

Qualquer atualização desta política será refletida nesta página. A data no topo indica a versão vigente. Mudanças significativas serão anunciadas nas notas de atualização do app.
